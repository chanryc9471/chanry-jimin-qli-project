Possible Inventions For the Future

Software
Virtual Builder

This is software that will allow you to make 3d models of architectural structures and can be used to project them using light on actual land. This is extremely helpful if you want to first see how your model will look before actually building it.

Find It

A software that can identify the material used on something with just a picture. Just like how an app can identify what plant it is by a snap of a picture, “Find It” can be used when you want to use material on a building. When you’re unsure what the material is, you could use the app to help you.

Hardware
Pocket Sketch

The “Pocket Sketch” is essentially a foldable tablet that is used to draw sketches or make 3d models of a design. The tablet has a library that stores all your sketches and 3d models up to 128 GB. You can always add more space with an SD card. The foldable design makes it so that the tablet won't take up much space. Attached to the tablet is a stylus pen specifically designed for the tablet, which adds to the paper-like type of screen. Another useful tool is the scale ruler feature, which allows the user to draw accurate marks. There is a variety of colors to choose from, so the sketches won’t be in plain black and white.
